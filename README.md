# Retool Product Analytics Platform

## Overview

This project simulates a modern B2B SaaS Product Analytics platform inspired by companies such as Retool.

The objective is to analyze customer behavior, product adoption, activation, retention, expansion, and churn using SQL, Python, Machine Learning, and Tableau.

---

## Business Problem

A SaaS company wants to answer questions such as:

- Why are customers churning?
- Which features increase retention?
- Which companies are most likely to expand?
- Which users become power users?
- Which workflows generate the most business value?

This project demonstrates how a Data Scientist can transform raw product events into actionable business insights.
